🔗 TRON Escrow System

Secure USDT escrow transactions on TRON blockchain

📋 System Components

🤖 Telegram Bot: User-friendly interface for creating escrow transactions

🔗 TronLink Interface: Web interface for signing transactions with TronLink wallet

📜 Smart Contracts: Solidity contracts handling USDT escrow logic

💳 TronLink Interface 📁 GitHub Repository

🔧 How to Use

  1. Start a conversation with the Telegram bot
  2. Create a new escrow transaction
  3. Enter recipient address and amount
  4. Click the TronLink button to open this interface
  5. Connect your TronLink wallet and confirm the transaction

🔒 Security Features

  • ✅ No private keys are ever stored or transmitted
  • ✅ All transaction signing happens in TronLink wallet
  • ✅ Smart contract enforces escrow rules automatically
  • ✅ Open source code for full transparency
